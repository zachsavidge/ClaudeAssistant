## Notes

- Some accounts may have slight name variations across different accounting systems. Use fuzzy matching or substring matching if an exact match isn't found.
- Bank account names (e.g., "三菱ＵＦＪ（法人）（API）大伝馬町支店") should be transliterated rather than literally translated. Use the English bank name + branch info.
- The "計" suffix means "total/subtotal" — always translate it as the parent category name + "Total".
